I. Visualizations -

The dashboard in AirTable counts each of the tags () and produces a
simple bar chart. Since this chart does not separate between X and Y, it
unintentionally overstates the I-9 audits, incidents of corporate
collaboration, worksite raids, home raids, public space raids, and ICE
sightings that have occurred up to 9.29.

For example, the visualization includes 970 ICE sightings -- but this
count includes sightings that have later been proven False (``Ice
Sighting, False'').
